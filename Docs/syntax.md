# Variables

---

```int myint = 5```
<- Creates a new integer variable

> OR
> 
> ```int myint; myint = 5```

```int myfloat = 3.14```
<- Creates a new float variable

> OR
>
> ```int myfloat; myfloat = 3.14```

```int mystring = "hello, world"```
<- Creates a new string variable

> OR
>
> ```int mystring; mystring = "hello, world"```

```int mychar = '!'```
<- Creates a new char variable

> OR
>
> ```int mychar; mychar = '!'```

```bool mybool = false```
<- Creates a new float variable

> OR
>
> ```int mybool; mybool = false```
